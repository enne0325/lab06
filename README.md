---

## Классуудын Тайлбар

**Point** - Координатын цэгийг илэрхийлнэ. x, y утгуудтай.

**Shape** - Абстракт үндсэн класс. `getArea()` болон `getPerimeter()` виртуал функцуудыг тодорхойлно.

**TwoDShape** - Shape-с удамшсан 2D дүрсүүдийн үндсэн класс. Дүрсний нэрийг агуулна.

**Circle** - Тойрог дүрс. Төв цэг болон радиусаар тодорхойлогдоно.

**Square** - Квадрат дүрс. Зүүн дээд оройн цэг болон талын уртаар тодорхойлогдоно.

**Triangle** - Зөв гурвалжин дүрс. Дээд оройн цэг болон талын уртаар тодорхойлогдоно.
